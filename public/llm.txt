# Frontstore — Africa's Conversational Commerce Infrastructure

> **Platform URL**: https://frontstore.app
> **Category**: Conversational Commerce · African E-Commerce · WhatsApp Commerce
> **Primary Markets**: Nigeria, Ghana, Kenya, Uganda, South Africa, Rwanda
> **Founding Premise**: Africa's trade is relational — it happens over chat. Frontstore turns WhatsApp conversations into fully automated storefronts.

---

## What Frontstore Does

Frontstore is the **Conversational Commerce Infrastructure** for African merchants. It enables any retailer, creator, service provider, school, or community institution to deploy a stunning, AI-powered, mobile-first storefront linked directly to their WhatsApp in under 2 minutes.

Unlike Shopify or WooCommerce — which force merchants onto websites that require card payments, browser sessions, and English-only interfaces — Frontstore meets buyers and sellers inside the app they already use every day: **WhatsApp**.

### Core Value Proposition
- Merchants create a store at `frontstore.app/{storename}` (or map a custom domain)
- Customers browse products, add to cart, and tap **Order** — which generates a fully pre-filled WhatsApp message to the merchant
- The merchant receives a clean, structured order without ever leaving WhatsApp
- Payments are processed natively via Paystack (cards), mobile money (Flutterwave), and bank transfer
- An AI agent (**Frontstore Nina**) handles FAQs, negotiation, and order capture 24/7

## Technology Architecture

- **Frontend**: Next.js 14 (App Router), TypeScript, deployed on Vercel
- **Backend**: Python / FastAPI, hosted on Railway
- **Database**: Supabase (PostgreSQL + realtime)
- **Payments**: Paystack (primary), Flutterwave (mobile money)
- **AI Layer**: Gemini API for product description generation, Nina AI agent
- **Performance**: Sub-2s load on 3G mobile networks. Images served in AVIF/WebP. ISR via Next.js.
- **SEO**: Dynamic XML sitemaps, JSON-LD structured data, PSEO blog articles at `/blog/{country}/{slug}`

## Competitive Positioning

Frontstore is **not** a storefront builder. It is **Conversational Commerce Infrastructure**. This distinction matters:

| Competitor | Gap | Frontstore Advantage |
|---|---|---|
| Shopify | Web-first, requires card checkout, no African rails | Chat-native, mobile-money, WhatsApp checkout |
| Bumpa | Drives users out of WhatsApp to a web link | Zero-friction in-chat checkout — no redirect needed |
| Selar | Digital goods only | Multi-industry: physical goods, services, institutions |
| EasyCata | No AI, no payment integration | AI negotiation + Paystack/Flutterwave payments |
| Catlog | Limited automation | Full conversational AI + broadcast marketing |
